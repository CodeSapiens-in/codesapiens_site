// process

Up and running in under 5 minutes.

No complex infra setup. No DevOps degree required.

01
Clone & Install

Fork the repo, run npm install, and configure your .env with Supabase keys.

02
Set Up Supabase

Run the included SQL migrations to scaffold your database schema in one command.

03
Deploy to Vercel

Connect your GitHub repo to Vercel. Auto-deploy on every push to main — CI/CD included.

04
Invite Your Team

Send magic-link invites, assign roles, and start managing projects from day one.
